# 🏇 LaLaLa式・究極の裏読みAI PRO

1番人気のオッズの歪みからレースの波乱度を判定し、最適な買い目を自動生成する競馬解析システムです。

## 🌟 主な機能
* **多次元AI解析**：5大オッズ（単勝・馬連・馬単・3連複・3連単）と世間の評判を入力し、独自のロジックでレースの「鉄板〜崩壊」度を4段階で判定。
* **オッズ歪み算出**：各馬券種のオッズ比率から、市場の歪みを数値化。
* **買い目自動生成**：直感印（◎○▲△×）と馬番を選ぶだけで、3連複BOX、馬連BOX、軸流しなどの買い目を自動計算。馬番の重複入力エラー防止機能付き。
* **資金管理システム**：予算と1点あたりの金額を入力することで、投資額を自動計算し、予算オーバーを警告。

## 💡 使い方
1. 左側のサイドバーに「5大オッズ」と「評判」を入力する。
2. 対象となる「馬番」をカンマ区切りで入力し、「LaLaLa印」を打つ。
3. 「資金（予算・1点金額）」を設定する。
4. 画面右側の「AI解析」でレース判定を確認する。
5. 「買い目生成」ボタンを押し、算出された買い目と投資額を確認する。
